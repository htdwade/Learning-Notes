## 1. Java下载安装

[jdk下载地址](https://www.oracle.com/technetwork/java/javase/downloads/index.html)

配置环境变量：

只需要配置bin目录到环境变量Path即可，不需要配置`ClASSPATH`：

```bash
JAVA_HOME E:\Java\jdk1.8.0_231
Path  %JAVA_HOME%\bin

//验证是否安装
java -version
```
