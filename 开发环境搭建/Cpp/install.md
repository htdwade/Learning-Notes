## 1. Mingw下载安装

[Mingw下载地址](https://sourceforge.net/projects/mingw-w64/files/)

下载`x86_64-posix-seh`离线包，解压，然后配置bin目录到环境变量Path中

```bash
Path E:\mingw64\bin

//验证是否安装
gcc -v
```
